# Pasindu Software Website

This is a premium, high-performance website for Pasindu Software.

## How to Run
Simply open `index.html` in any modern web browser. No installation is required.

## Features
- **Modern Design**: Dark mode, glassmorphism, and smooth animations.
- **Responsiveness**: Works perfectly on mobile, tablet, and desktop.
- **Interactive Elements**: Hover effects, scroll animations, and filtered product lists.

## File Structure
- `index.html` - Home page
- `products.html` - Software catalog with filtering
- `about.html` - Company information and team
- `contact.html` - Contact form and details

## Technologies
- HTML5
- Tailwind CSS (via CDN)
- Google Fonts (Outfit)
- JavaScript (Vanilla)
